LockBox

Overview

LockBox is a user-friendly desktop application designed to secure data through encryption and expiration settings. It targets users with limited cybersecurity knowledge, offering an intuitive interface for secure data handling.

Contributions

  • Enabled users to set expiration dates for shared data.

  • HTML/QSS
